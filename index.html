The Problem

Meet Hector

Hector has an hearing disability.

He hears louder on his right side than his left side.

Hector boughts a special sound system to solve his problem, but it was badly wired from the backside.

Now we need to fix it for him, by software

The solution (pseudocode)

The stereo has these predefined values for the volume level of each speaker:

int left = 30;
int right = 70;

How can we solve this?

The most simple solution can be done in 3 steps

  1. save the left value to a temp variable
  2. modify the left value to the value of right
  3. modify the right value with the value in temp variable

(DEMO)

Using a simple swap algoritm
Main.javapackage org.academiadecodigo.bootcamp;
public class Main {
	public static void main(String[] args) {

		int left = 30;
		int right = 70;

	}
}
Main.javapackage org.academiadecodigo.bootcamp;
public class Main {
	public static void main(String[] args) {
		int left = 30;
		int right = 70;

		int temp = left;
		left = right;
		right = temp;

		System.out.println("left: " + left + " right: " + right);
	}
}
> left: 70 right: 30

ALLRIGHT!!!!

Why this isn't working?
